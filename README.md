# Professional Portfolio

A clean, single-page portfolio website with Home, About Me, Projects, and Contact sections.

## Quick start

Open `index.html` in your browser, or run a local server:

```bash
npx serve .
```

## Customize

1. **Name & title** — Edit the hero section in `index.html` (`Your Name`, role, intro).
2. **Social links** — Update GitHub, LinkedIn, X, and email URLs in the Home section and footer.
3. **About** — Replace bio text, skills, and detail list (location, education).
4. **Projects** — Swap placeholder cards with your real projects, demo links, and repos.
5. **Contact** — Set email, phone, and location; connect the form to [Formspree](https://formspree.io) or Netlify Forms for live submissions.

## Resume / CV

Click **Resume / CV** on the home page, or open `js/pages/resume.html` directly. The resume uses a Harvard-style (single-column, conservative) layout with dummy IT graduate data — replace names, dates, skills, and experience with your own. Use **Print / Save PDF** in the browser to export.

## Files

- `index.html` — Structure and content
- `css/styles.css` — Portfolio layout and design
- `js/script.js` — Navigation, scroll effects, mobile menu
- `js/pages/resume.html` — Resume / CV template (Harvard style, dummy data)
- `css/resume.css` — Resume styling and print layout
- `assets/profile.png` — Profile photo (portfolio home page)
